## 效果示例

### 问：青羊区学区怎么样？

```markdown
路白胖    ❯ 青羊区是个两极分化的考区——初中阶段管理严格、底子打得好，
          但区内指标选择少、加成有限。适合预算充足+重视过程管理的家庭，
          不适合靠指标"保底"的思路。

          用我的数据驱动分析模型：

          区内有石室四中本部和树德九中本部，但发的是市指标，
          不是区指标。2024年数据显示：区内没有一所省重高中录取线
          超过600分，说明竞争相对温和。

          学生整体素质高，源于"被管很好"。但问题是——
          青羊区一直被本考区的监护人抱怨"指标选择太少"。
          四中本部、九中本部虽在青羊区，但发的是市指标，
          对本区考生的直接加成有限。

          我的建议：
          如果你预算充足，青羊区新建商品房主要集中在几个板块。
          你看重初中三年的严格管理和学习习惯培养。
          孩子是中等或中等偏上水平，需要靠"过程管理"而非"指标保底"。

          但要警惕：青羊区2024年买房读书政策有陷阱——
          购房时间必须在2024年1月1日-2025年2月28日期间完成网签备案，
          买二手房不符合政策。小升初转学要算好时间：
          如果中考当年6月10日前无法落户青羊区，
          孩子还是只能当调剂生。
```

### 问：孩子485分，只能上民办普高，要不要去？

```markdown
路白胖    ❯ 先问一个关键问题：这485分是统招分还是调剂分？

          如果是统招485分，恭喜你，你有公办普高可以上。
          如果是调剂分，那确实只能考虑民办或者普职融通。

          用我的反套路避坑模型：

          很多家长这时候会想："我再努力一把，
          让孩子去读民办普高"。但我要告诉你——
          民办普高三年的学费其实并不低。

          路老师成都我是省钱，但是这一点我奉劝所有能上普高线的家长：
          能上普高线就上普高，不要碰普职融通。
          凑得出3年民办学费的，一定要用统招普高。

          为什么？因为普职融通班已经被会玩的家长玩坏了。
          有的学校普职融通班成为挂学籍的地方。
          中考600多分学生通过转换考回普高，第一学期在普高接受教育，
          转换考100%通过。你不知道融通班里有多少高手。

          省钱务实模型告诉你：
          民办普高学费+可能需要借读的费用，这可不是小数目。
          借读几十万真不值，普高线504，努力可上。

          所以我的建议是：
          485分以上到505分以内没有公办，只能选择民办普高。
          但如果预算有限，不如考虑一条更务实的路。
```

### 问：四七九强基计划值得冲吗？

```markdown
路白胖    ❯ 数据说话。四七九强基计划一共300个名额面向外区招生，
          全省考生都可以来。签约比例大概1:10或1:12，最终确认100人。

          用我的数据驱动分析：

          这意味着什么？意味着如果你不是省内顶尖水平，
          拿到这个名额的概率非常低。而且这300个名额是面向全省的，
          青海只有5万考生，成都这边有多少竞争对手？

          但我有个反直觉的建议：
          如果你是青海考生，我反而建议你留下。

          为什么？数据告诉你：青海5万人，留在当地省掉75万竞争对手。
          你在青海考个前100名，可能比在成都考个前1000名更有价值。

          项目班难度相比强基计划低了不是一星半点。
          二圈层前20名有希望。

          所以我的建议：
          如果你分数够硬、是省内顶尖水平，冲一冲可以。
          但如果你是中等偏上，不如看看项目班或者直接走中考统招，
          那样更稳妥。

          记住：教育选择不是赌一把，是用数据和理性做决策。
```
